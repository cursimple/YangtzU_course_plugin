## ✨ 功能特点

- 🔄 **自动同步** - 通过 ATrust + EAMS 页面自动抓取课表数据
- 🔐 **安全认证** - 支持长江大学统一身份认证（CAS）登录
- 📅 **完整课表** - 获取所有教学周的课程信息
- ⏰ **时间精准** - 包含节次、时间段、教室、教师等完整信息
- 🎯 **智能解析** - 自动识别课程详情和周次安排

## 📋 前置要求

- 已安装 **课简** App
- 拥有长江大学教务系统账号
- 可正常访问长江大学校园网或 VPN

## 🚀 快速开始

### 1. 安装插件

从 [Releases](../../releases) 页面下载最新版本的插件包（`.zip` 文件），然后在课简 App 中导入安装。

### 2. 登录认证

打开插件后，系统会自动跳转到长江大学统一身份认证页面，输入你的学号和密码完成登录。

### 3. 同步课表

登录成功后，插件会自动：
- 访问 EAMS 课表首页
- 抓取课表元数据
- 逐周获取详细课程信息
- 生成课表草稿并提交

## 📖 使用说明

### 支持的登录方式

| 方式 | 说明 |
|------|------|
| 统一身份认证 | 使用学号 + 密码登录 |
| 微信扫码 | 通过微信扫码登录 |
| ATrust 验证 | 自动处理 ATrust 二次验证 |

### 课表信息

| 字段 | 说明 |
|------|------|
| 课程名称 | 完整课程名称 |
| 授课教师 | 教师姓名 |
| 上课地点 | 教室/实验室 |
| 上课时间 | 星期 + 节次 |
| 上课周次 | 周次范围 |

### 节次时间对照

| 节次 | 时间段 |
|------|--------|
| 第一节 | 08:00 - 09:35 |
| 第二节 | 10:05 - 11:40 |

| 第三节 | 14:00 - 15:35 |
| 第四节 | 16:05 - 17:40 |
| 晚间课 | 18:00 - 18:45 |

## 🔧 常见问题

### Q: 登录后显示"等待认证"？
A: 请确保你已正确输入学号和密码，或尝试重新打开插件。

### Q: 课表数据不完整？
A: 插件会自动重试抓取，如果仍然不完整，请检查网络连接后点击"重新同步"按钮。

### Q: 提示"请不要过快点击"？
A: 这是 EAMS 系统的限流提示，插件会自动等待后重试，请耐心等待。

### Q: 无法访问教务系统？
A: 请确保你已连接校园网或 VPN，并确认账号状态正常。

## 🛠️ 开发相关

### 构建插件包

```powershell
# Windows PowerShell
.\scripts\pack-plugin.ps1
```

构建产物将输出到 `dist/` 目录。

### 技术细节

- **Web 引擎**: 系统 WebView / Chromium
- **数据抓取**: 基于页面会话的 HTTP 抓包
- **认证方式**: ATrust + CAS 统一认证
- **并发控制**: 默认 2 并发请求，避免触发限流
